MIPS Assembly Language

Centipede - 2021

Project link here

This is a final project from my 2nd year computer organization course: CSC258. The purpose of this project is to create a modified version of the 1980 game Centipede using MIPS assembly.

The controls are:

  • j - Move bug blaster to the left
  • k - Move bug blaster to the right
  • x - Shoot a dart from the bug blaster
  • s - Restart the game
  • q - Quit

The features implemented in this program are:

  • Only one dart on the screen at a time
  • The centipede dies when a dart hits it 3 times
  • Only one flea falls down from the top at a time (at random place)
  • Game over when the flea intersects the bug blaster
  • Mushrooms disappear when dart hits them
  • A new centipede spawns from the top when a centipede dies
  • Centipede continues wriggling near the bottom of screen when it reaches the bottom
